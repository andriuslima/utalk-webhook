## Dependencies

1. Install `nvm`: https://github.com/nvm-sh/nvm#installing-and-updating
2. Install `yarn`: https://classic.yarnpkg.com/lang/en/docs/install/

## Running the app

First run `nvm use` in the project directory

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
